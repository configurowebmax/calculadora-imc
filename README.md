# Calculadora de IMC Gratis

Aplicación web gratuita para **calcular tu Índice de Masa Corporal (IMC)** y conocer tu rango de peso saludable.

## ✨ Características

- Cálculo instantáneo del IMC.
- Clasificación por colores (bajo peso, normal, sobrepeso, obesidad).
- Rango de **peso saludable** según tu estatura.
- Tabla de referencia.

## 🚀 Uso

1. Ingresa tu **peso** (kg) y **estatura** (cm).
2. Pulsa **Calcular IMC**.
